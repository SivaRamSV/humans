# 🌌 Cosmic Timeline

An interactive visualization of cosmic history — from the Big Bang to humans.

**Motivation:** Inspired by *The Dinosaurs* (2026) on Netflix — a documentary series narrated by Morgan Freeman that explores the 165-million-year reign of dinosaurs. After watching it, I wanted to visualize the full timeline of life on Earth and see just how recent human existence is on the cosmic scale.

Best viewed on desktop browsers. I tried my best to make it look good on mobile too!

## Tech Stack

- React 19 + TypeScript
- Vite
- React Three Fiber (Three.js)
- Framer Motion
- Zustand
- Tailwind CSS 4
